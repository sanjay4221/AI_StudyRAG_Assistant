# Contributing to StudyRAG

Thank you for your interest in contributing! Here's how to get started.

---

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/sanjay4221/AI_StudyRAG_Assistant.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Set up your local environment (see [README.md](README.md))
5. Make your changes
6. Run tests: `cd backend && pytest`
7. Commit and push
8. Open a Pull Request

---

## Development Guidelines

- Follow existing code style and naming conventions
- Add tests for new features (aim for coverage in the `tests/` folder)
- Update `CHANGELOG.md` with your changes
- Keep PRs focused — one feature or fix per PR

---

## Reporting Issues

Open a GitHub Issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (OS, Python version, browser)

---

## Feature Requests

Open a GitHub Issue with the `enhancement` label. Describe the use case and why it would benefit students.
